## 説明
積み上げないでください。<br>
こちらから遊べます。⇒  [WebMSX](http://webmsx.org/?MACHINE=MSX1J&PRESETS=MSXMUSIC&ROM=https://github.com/CoBinee/uptack-msx/raw/main/rom/UPTACK.ROM)

## 操作
- `←`, `→`<br>左右に移動
- `SPACE`<br>時計回りに回転
- `SHIFT`<br>反時計回りに回転
- ジョイスティックにも対応

## 動作環境
- 16KB 以上の RAM を搭載した MSX

## 開発
- [SDCC](https://sdcc.sourceforge.net)
